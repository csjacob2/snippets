Breaking A Weak AES Key

We encrypted a Jeopardy clue using 256-bit AES in CBC and stored the result in hexadecimal in the file aes_weak_ciphertext.hex. We haven't supplied the key. All we'll tell you about the key is that it is 256 bits long and its 251 most significant (leftmost) bits are all 0's. The initialization vector was set to all 0s. Find the plaintext.

Note: the decryption is cut off as the assigned ciphertext was actually cut off as well (unintentionally).
